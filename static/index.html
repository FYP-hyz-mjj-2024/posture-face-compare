Greetings! But the frontend does not run on this server!

It runs locally with the inference backend.

The frontend is a locally run project (https://github.com/FYP-hyz-mjj-2024/posture-video-stream) written in Next.js and React framework. Please take a peek as the below image:

This server is the Storage Backend of the Pedestrian Cell Phone Usage Detection System (alias, You Focus Your Walk), which serves solely for API function handling and face storage.

The frontend is intended to run locally to meet the WebSocket throughput requirements of the inference backend. For more information, please refer to the GitHub repositories of the three components:

  • Inference Backend
  • Frontend (Use this if you want to access the UI)
  • Storage Backend (This server)
